# Kind Hands — Firebase + React Application

Kind Hands is a web application built with **React (Vite)** and **Firebase**.  
It uses **Firebase Authentication**, **Firestore**, and supports **Firebase Hosting** for deployment.

---

## ✅ Features
✅ User registration (email/password)  
✅ Login & logout  
✅ Role-based user document stored in Firestore  
✅ Protected routes using React Router  
✅ Firebase Hosting support  
✅ Environment variables for secure config

---

## ✅ Requirements
- Node.js **20+**
- npm
- Firebase project (console.firebase.google.com)

## ✅ 1. Clone & Install
```bash
git clone <REPO_URL>
cd kind_hands_app
npm install
```

### If React/Vite aren’t installed yet:
```bash
npm install react react-dom
npm install -D vite @vitejs/plugin-react-swc
```

## Start the Vite dev server:
```bash
npm run dev
```

## Build production files:
```bash
npm run build
```

## Preview production build:
```bash
npm run preview
```
